# 📝 Text Rephraser - Интеллектуальная обработка текстов

**`Text Rephraser`** — это многофункциональное приложение на языке Python 🐍 для интеллектуальной обработки текстовых документов. Приложение сочетает мощные возможности извлечения текста из различных форматов с гибкой системой управления настройками и современным веб-интерфейсом. Инструмент идеально подходит для исследователей, студентов и профессионалов, работающих с научными текстами. ✨

---

## 🌟 Основные возможности

- 📂 **Многоформатная поддержка**: Работа с файлами `.pdf`, `.txt`, `.md` и `.docx`
- 📖 **Интеллектуальная обработка**: Переформулировка текста с сохранением научной терминологии
- 🎚️ **Настройка уровня перефразирования**: Гибкий контроль степени изменения текста (temperature)
- 🖱️ **Гибкий ввод данных**: Аргументы командной строки или интерактивный режим
- 🎨 **Автоматическая иллюстрация**: Обновление и генерация изображений в PDF (в разработке)
- 🛠️ **Расширенное логирование**: Детальная диагностика и отладка процессов
- 🛡️ **Надежная обработка**: Устойчивость к различным типам ошибок
- 🌐 **Современный интерфейс**: Веб-приложение для удобной работы

## 🚀 Установка и запуск

### 1. Установка зависимостей

   ```bash
   pip install -r requirements.txt
   ```

### 2. Настройка API-ключей

#### OpenAI API
Укажите ключ в переменной `OPENAI_API_KEY` в файле `main.py` или введите через интерфейс.

#### Дополнительные API (для иллюстраций)
- **NanoBanana API**: Для генерации изображений
- **Google Custom Search API**: Для поиска клинических изображений

```bash
python run_web_app.py
```

Приложение будет доступно по адресу: **http://localhost:8501**

### 4. Альтернативный запуск

```bash
# Прямой запуск через Streamlit
streamlit run app.py

# Командная строка
python main.py --input-file input/file.pdf --theme "ТЕМА"
```

## 🎚️ Функция настройки уровня перефразирования (Temperature)

### Описание
Гибкая настройка степени творческого переформулирования текста в диапазоне от 0.0 до 1.0.

### Режимы работы

1. **📌 Консервативный (0.0-0.3)**
   - Максимальное сохранение оригинальной структуры
   - Минимальные изменения в тексте
   - Точное сохранение терминологии
   - Подходит для: строго научных текстов, официальных документов

2. **⚖️ Сбалансированный (0.4-0.6)**
   - Умеренное перефразирование
   - Сохранение академического стиля
   - Баланс между точностью и вариативностью
   - Подходит для: научных статей, исследований, учебных материалов

3. **✨ Творческий (0.7-1.0)**
   - Более вариативное переформулирование
   - Сохранение смысла при большей свободе выражения
   - Разнообразие формулировок
   - Подходит для: популяризации науки, адаптации текстов

### Использование

#### В веб-интерфейсе
1. Перейдите на вкладку "🔄 Перефразирование"
2. Используйте слайдер "Уровень творческого переформулирования"
3. Выберите значение от 0.0 до 1.0
4. Загрузите файл и начните обработку

#### В командной строке
```bash
# Консервативный режим
python main.py --input-file input/file.pdf --temperature 0.2 --theme "МЕДИЦИНА"

# Сбалансированный режим (рекомендуется)
python main.py --input-file input/file.pdf --temperature 0.5 --theme "ФИЗИКА"

# Творческий режим
python main.py --input-file input/file.pdf --temperature 0.8 --theme "БИОЛОГИЯ"
```

### Рекомендации по выбору

| Тип текста | Рекомендуемая temperature | Обоснование |
|------------|---------------------------|-------------|
| Медицинские документы | 0.1 - 0.2 | Критична точность терминов |
| Научные статьи | 0.3 - 0.5 | Сохранение стиля, умеренная вариация |
| Учебные материалы | 0.4 - 0.6 | Ясность изложения |
| Популярная наука | 0.6 - 0.8 | Доступность, вариативность |
| Адаптация текстов | 0.7 - 0.9 | Максимальная вариативность |

## 🎭 Функция перерисовки изображений через DALL-E 2

### Возможности
- **Интеграция с DALL-E 2**: Перерисовка изображений через OpenAI DALL-E 2 API
- **Интерактивный выбор**: Выбор изображений из извлеченных из PDF
- **Редактирование промптов**: Настройка текстового описания для генерации
- **Гибкие настройки**: Выбор размера (256x256, 512x512, 1024x1024) и качества
- **Управление результатами**: Возможность утвердить, перегенерировать или скачать

### Настройка DALL-E 2 API
1. **API-ключ OpenAI**: Используется тот же ключ, что и для перефразирования текста
2. **Доступ к DALL-E 2**: Убедитесь, что ваш аккаунт OpenAI имеет доступ к DALL-E 2
3. **Настройка в интерфейсе**: Введите API-ключ в разделе "⚙️ Настройки" → "🎭 DALL-E 2 API"

### Использование

#### В веб-интерфейсе
1. Перейдите на вкладку "🎭 Перерисовка"
2. Выберите изображение из списка извлеченных
3. Настройте параметры генерации:
   - **Размер**: 256x256, 512x512, 1024x1024
   - **Качество**: standard или HD (только для 1024x1024)
   - **Промпт**: Описание желаемого результата
4. Нажмите "🎭 Перерисовать"
5. Оцените результат и выберите действие:
   - **✅ Утвердить**: Сохранить изображение
   - **🔄 Перегенерировать**: Создать новый вариант
   - **📥 Скачать**: Скачать изображение

#### Особенности работы
- **Автоматические промпты**: Если промпт не указан, система генерирует его на основе контекста изображения
- **Медицинский стиль**: По умолчанию используется медицинский стиль иллюстраций
- **Сохранение результатов**: Перерисованные изображения сохраняются в папку `extracted_images/`
- **Стоимость**: DALL-E 2 тарифицируется отдельно от GPT API

### Примеры промптов

```
"Medical X-ray showing bone fracture. Professional diagnostic image, clear anatomical details, educational for medical students."

"Modern medical illustration of human skeletal system. Clean lines, anatomical accuracy, educational diagram style."

"Clinical imaging of osteoporosis in spine. Professional healthcare visual, clear and informative."
```

### Технические требования
- **API-ключ**: Действующий ключ OpenAI с доступом к DALL-E 2
- **Квота**: Достаточная квота на генерацию изображений
- **Изображения**: Наличие извлеченных изображений в папке `extracted_images/`

## 🛠️ Командная строка

### Основные аргументы

```bash
python main.py [OPTIONS]

Options:
  --input-file PATH         Путь к входному файлу (.pdf, .txt, .md, .docx)
  --output-file PATH        Путь к выходному файлу (.txt) [по умолчанию: output/paraphrased.txt]

  --temperature FLOAT       Уровень перефразирования (0.0-1.0) [по умолчанию: 0.4]

  --show-settings           Показать текущие настройки
  --help                    Показать справку
```

```bash

python main.py --input-file input/book.pdf --theme "МЕДИЦИНА"

# С настройкой температуры
python main.py --input-file input/article.pdf --temperature 0.3 --theme "ФИЗИКА"

# С сохранением API-ключа
python main.py --set-api-key "sk-your-api-key-here"

# Просмотр настроек
```

## 🆘 Решение проблем

### Проблемы с запуском

#### "ModuleNotFoundError"
```bash
pip install -r requirements.txt
```

#### "API ключ не настроен"
- Введите ключ в веб-интерфейсе на вкладке "⚙️ Настройки"
- Или используйте `python main.py --set-api-key "ваш-ключ"`

#### "Не удалось извлечь текст из файла"
- Проверьте формат файла (PDF, TXT, MD, DOCX)
- Убедитесь, что файл не поврежден

### Проблемы с temperature

#### "Значение не сохраняется"
```bash
# Проверьте права на запись
ls -la settings.json
cat settings.json
```

#### "Результаты одинаковые при разных temperature"
- API может давать похожие результаты на коротких текстах
- Попробуйте на большем фрагменте текста

### Проблемы с иллюстрациями

#### "API ключ NanoBanana не настроен"
- Получите ключ на nanobanana.ai
- Введите в разделе настроек

#### "Ошибка поиска в TCIA"
- Проверьте интернет-соединение
- TCIA API может быть временно недоступен
